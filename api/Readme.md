# Public Supply Endpoints Documentation

This document explains the public API endpoints for retrieving supply-related data of the AUN token.

## Base URL
https://api-prod.aurumnetwork.io

## Endpoints

### GET api/v1/aun/total-supply
Retrieves the total supply of AUN tokens.

#### Example Request
```http
GET api/v1/aun/total-supply
```

#### Example Response
```json
999999327.0876821
```

### GET api/v1/aun/circulating-supply
Retrieves the circulating supply of AUN tokens.

#### Example Request
```http
GET api/v1/aun/circulating-supply
```

#### Example Response
```json
69600000
